  1. Question: What are the three components of Web?

    Answer: HTML, CSS, and JavaScript

  2. Question: Define version control and give one example.

    Answer: Version control is a system that records changes to something over time so that you can easily revert to an older version as needed. Usually done through software like GitHub.

  5. Question: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason.

    Answer: The odd one out is <a> which only deals with linking URLs, while the other three deal with content formatting.

  6. Question: What are the tags required to create a navigation list in HTML?

    Answer: Primarily you need a <nav> tag

  7. Question: What element is used to create images? Name its attributes.

    Answer: The element itself is <img>, and the attributes for it are src (which is the most important, as it's the URL of the image), alt, height, width, alt, longdesc, and ismap. There are other attributes, such as align and border, but they are not supported in HTML5

  8. Question: Identify the odd one from these tags: <table>, <tr>, <b>, <td>, and state the reason.

    Answer: The odd one out is <b>. The other three are used for tables, while <b> is only used for bold text

  9. Question: Give an example of color picker input.

    <input type="color" name="co" id="co">

  10. Question: What is the use of name and value attributes? Give two examples

    Answer: name is exactly what it says on the tin: it's the name of an element, which is used for reference when JavaScript and CSS start getting added to the code.
    value is similar, but provides a value for an element, and isn't used for reference, but for input purposes, or increment counting for <li> tags.
    Examples:
    <input type="checkbox" name="places" id="france">
    In this example, the name is "places," which can be used for other checkbox elements
    <li value="2">
    In this example, if an <ol> is used in the <li> element, it will start with 2, because that's what the value is
